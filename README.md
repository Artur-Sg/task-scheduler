# Task Scheduler

Платформенно-независимая библиотека планирования задач с поддержкой приоритетов, отложенного выполнения, кооперативного исполнения, отмены задач и подключаемых исполнителей.

## Мотивация

Современные приложения регулярно выполняют ресурсоёмкие операции:

- фильтрацию больших наборов данных;
- обработку файлов;
- вычисление производных данных;
- анализ и преобразование структур данных;
- фоновую обработку пользовательского ввода.

Выполнение таких операций без явной стратегии планирования может блокировать основной поток исполнения и ухудшать отзывчивость приложения.

## Цель проекта

Цель проекта — исследовать и реализовать универсальную модель планирования вычислительной работы, позволяющую управлять порядком, приоритетом и способом выполнения задач.

Управление задачами осуществляется через:

- приоритетное планирование;
- отложенный запуск;
- кооперативное исполнение длительных задач;
- отмену задач;
- подключаемые исполнители;
- возможность вынесения тяжёлой работы во внешние исполнители, например Worker-пул.

## Основные сценарии использования

### Поиск по большим наборам данных

Пользователь продолжает вводить текст, пока выполняется фильтрация данных.

### Импорт файлов

Обработка больших файлов выполняется в фоне и может быть отменена пользователем.

### Вычисление производных данных

Фоновые вычисления не должны мешать пользовательскому взаимодействию.

### Вынесение тяжёлой работы

Тяжёлые вычисления могут быть вынесены во внешний исполнитель, чтобы не блокировать основное приложение.
